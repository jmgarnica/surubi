Composcion del proyecto
-> CMPTest: proyecto de UnitTest de .Net
	->  test.json: tests utilizados pras probar el compilador
-> Informe
-> Surubi: proyecto para la interfas de consola del compilador
-> TigerCs: biblioteca de clases que componen el compilador
-> NASM.rar: contiene los archivos necesarios para obtener un ejecutable, debe se copiada a la carpeta donde se ejecutara el compilador y el tester.
